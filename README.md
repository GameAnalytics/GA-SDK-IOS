GA-SDK-IOS
==========

GameAnalytics native iOS SDK.

Documentation in the [wiki](https://github.com/GameAnalytics/GA-SDK-IOS/wiki).

Read more about our launch of the new events (V2) [here](http://www.gameanalytics.com/update/).

> :information_source:
> Requirements:<br/>
> **iOS:** iOS 6.x and up
